Front-End
html 간단한 HTML 구조 설계 및 Timeleaf를 활용하여 HTML과 Controller에서 활용하는 Java 문법과의 연동이 가능하며 적절한 class 부여가 가능합니다.
css HTML 구조에 따라 적절한 위치 조정 및 색감 조절, 속성의 기본 디자인 변경 등이 가능합니다.
class 값에 따른 적절한 속성 부여와 hover 및 active와 같은 기능에 따른 디자인 변경이 가능합니다.
java-script Javascript를 활용하여 HTML 파일에서 생성한 요소(input 등)를 통한 이벤트를 설정 가능하며 AJAX(xhr)를 활용한 비동기적 통신에 따른 결과를 도출할 수 있습니다.
또한, Javascript에서 HTML 문법과의 연동을 통해 넘겨준 값을 적절하게 사용할 수 있으며 상황에 알맞은 함수 구현과 활용이 가능합니다.
Back-End
java 자바 기초 및 자료구조에 대한 이해도 있음. 객체지향 문법에 대한 이해도가 존재함. 알고리즘 문제풀이를 통해 Java에 대한 이해도가 있습니다.
Spring Boot를 사용하여 웹 개발하는 것에 있어 원활한 구현을 가능케 합니다.
접근자(Public, Private, Default, Protected)를 잘 이해하고 있으며 적절한 사용이 가능합니다.
spring-boot Java와 DataBase를 활용한 웹 개발이 가능합니다.
회원가입, e-mail 인증, 주소 찾기, 게시판 만들기 및 게시글 작성, 댓글 작성, 대댓글 작성 등 CRUD 기능 구현이 가능합니다.
또한, CRUD 기능 수행 작업을 수행하는 것에 있어 차질 없는 업무 수행이 가능함.
Controller, Service, IMapper 인터페이스, Mapper 간의 관계와 어노테이션을 활용하여 적절한 연동을 가능케 함.
database SQL 문법에 대한 이해도가 있으며 주로 MariaDB를 사용하며, JDBC, MyBatis를 활용하여 DB와 Java의 연동이 가능함.
기본키, 외래키, 유니크 키에 대해 정확히 이ㅎ하고 있으며, 적절한 사용이 가능함.
JOIN 역시 충분한 이해도가 있으며 상황에 따른 적절한 활용이 가능함.
